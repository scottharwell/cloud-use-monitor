Simple scripts to help get info for Azure use.

## Setup

Set the following environment variables so that the scripts can acquire an Oauth token to operate against the Azure APIs.

* `CLIENT_ID`
* `CLIENT_SECRET`
* `SUBSCRIPTION_ID`
* `TENANT_ID`
* `REGION`
* `MINS_TO_RUN`

## Scripts

1. `container_instance.py` Outputs the use of container instance data from an Azure Region.
